## 🚀 Featured projects

- **[AI Explorer Pharmacy](https://github.com/DPS1031/AI_Explorer)** — AI-powered pharmacy assistant built with Python, Streamlit, OpenAI GPT-4o-mini, and PostgreSQL. Converts natural language to SQL, identifies medications from photos, handles conversational ordering, and emails PDF invoices via AWS SES. Deployed on EC2 with Docker Compose and private ECR repos.
- **[Omni-Commerce](https://github.com/DPS1031/ecommerce-project)** — Full-stack e-commerce platform with FastAPI and Angular. JWT HttpOnly cookie auth, Flyway-versioned PostgreSQL migrations, dual Angular clients (storefront + admin), and full Docker orchestration.
- **[Delivery Appointment Management System](https://github.com/DPS1031/Delivery-Appointment-Management-System)** — Production-ready fullstack app with Django REST Framework and Next.js (App Router, TypeScript). JWT auth, state-machine-based status management, real-time reports with raw SQL, and CI/CD via GitHub Actions.
- **[Products CRUD Full Stack](https://github.com/DPS1031/products-crud-fullstack)** — TypeScript app with a Node.js/Express backend and React frontend, deployed on AWS EC2 with multi-stage Docker builds and an Nginx reverse proxy.
- **[Inventory Management System](https://github.com/DPS1031/Managment-inventory-web)** — Built with ASP.NET Core and Entity Framework. JWT authentication and multi-level hierarchical data management on SQL Server.
- **[Responsive Dashboard](https://github.com/DPS1031/MyDasboardApp)** — React + TypeScript dashboard with secure JWT authentication and real-time data visualization using ApexCharts, containerized and served through Nginx.
- **[Angular Product Showcase](https://github.com/DPS1031/Angular-product-showcase)** — Standalone Angular components with reactive state management and polished UI interactions built with Tailwind CSS.
- **[Class Diagram Generator (C#)](https://github.com/DPS1031/ClassDiagram_CSharp)** — C# project for modeling and generating class diagrams.
- **[Billing System (PHP)](https://github.com/DPS1031/BillingSystem-php)** — PHP-based billing system.
- **[Web Project (PHP)](https://github.com/DPS1031/Webproject-php)** — WordPress/PHP site work, focused on UI improvements and REST API integrations.
